Data

This page contains the data as ready to use csv files. Replication code used for MONSTr and all related publications is publicly available at our downloadable github repository.

Data summary

The following table provides a summary of the variables coded for each military operations in MONSTr. Variables derived from those listed (logged, binary, etc) are not shown.

Type Variables
Parent War, ICB, IMI, MID, MIP, MIPS, RAND
Time Start date, End date, Duration, Parent start date, Days into parent
Means Drones, Air-to-air, Cruise missiles, Aerial bombing, Close air support, Ground troops, Paramilitary
Participants US allies, Side B state belligerents, Side B non-state belligerents
Geography Latitude-longitude, Urban, Forested, Mountainous
Casualties US, Side A, Side B, Civilian
Domestic US President, Political party, Election year, Approval rating
